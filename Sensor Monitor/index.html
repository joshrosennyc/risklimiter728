How Does Sensor Monitor Work?

Read below and watch this short video to learn more.

  • RiskLimiter sensors are mounted to the controlled cooler or freezer. Sensor can be placed inside or outside of the cooler or freezer.
  • Sensors are programmed to record temperatures at any interval desired.
  • Data is first communicated wirelessly to the RiskLimiter Ethernet gateway, placed up to 1,200 feet away, and then transmitted to the RiskLimiter cloud in real time, without the need for a computer.
  • Out-of-safe-range alerts are sent in real time to designate personnel via email, text, or the RiskLimiter Manager’s Application so corrective action can be taken immediately.
